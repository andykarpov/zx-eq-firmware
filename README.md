# ZX-EQ Firmware

Альтернативная прошивка под проект ZX-EQ (https://github.com/RBSC/ZX-EQ).

1) zx_eq/zx_eq_10bit_pos, zx_eq/zx_eq_10bit_neg - версия под светодиодные столбики из 10 светодиодов;
2) zx_eq/zx_eq_8bit_pos, zx_eq/zx_eq_8bit_neg - версия под столбики из 8 светодиодов;
3) zx_eq/msx_eq_10bit_pos, zx_eq/msx_eq_10bit_neg - оригинальная автораская прошивка под столбики из 10 светодиодов.
4) zx_eq/msx_eq_8bit_pos, zx_eq/msx_eq_8bit_neg - оригинальная авторская прошивка под столбики из 8 светодиодов.

Все новые прошивки рассчитаны под светодиодные столбики, у которых либо общий катод (суффикс _neg), либо общий анод (суффикс _pos).
