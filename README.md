# ZX-EQ Firmware

Альтернативная прошивка под проект ZX-EQ (https://github.com/RBSC/ZX-EQ).

1) 10bit - версия под светодиодные столбики из 10 светодиодов;
2) 8bit - версия под столбики из 8 светодиодов;
3) orig - оригинальная автораская прошивка под столбики из 10 светодиодов.

Все 4 новые прошивки (2 под 8bit и 2 под 10bit) - под столбики, у которых либо общий катод (суффикс _neg), либо общий анод (суффикс _pos).
Оригинальная прошивка, по схеме - с общим анодом (_pos).
